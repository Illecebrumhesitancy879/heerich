## 🧭 Overview

heerich is a small app for rendering 3D voxel scenes as SVG files on Windows.

It is made for simple scene building, shape checks, and clean image output. You can use it to view voxel art with oblique or perspective cameras, then export the result as SVG.

The app uses no extra dependencies, so setup stays light. It fits users who want a direct way to open a scene and render it.

## ✨ What it does

- Build 3D voxel scenes
- Render scenes to SVG
- Use boolean ops for shape work
- Switch between oblique and perspective views
- Keep output clean and easy to share
- Run with zero extra libraries

## 🎛️ Basic use

### Add voxels

Use the scene tools to place voxel blocks in the workspace. Build small forms first so you can see how the scene behaves.

### Shape operations

Use boolean ops to combine or cut shapes. This helps when you want clean edges or simple sculpted forms.

### Camera modes

heerich supports two main camera styles:

- Oblique projection for a graphic, technical look
- Perspective for depth and a more natural view

Try both modes and keep the one that fits your scene.

### Export to SVG

When the scene is ready, render it to SVG. SVG files are easy to view in browsers and can be opened in many design tools.

## 🛠️ Troubleshooting

### The app does not open

- Check that the file finished downloading
- Extract the zip file first
- Try running the app again
- Right-click the file and choose Run as administrator

### The scene looks wrong

- Change the camera mode
- Check the voxel placement
- Remove shapes that overlap in the wrong place
- Render again after each change

### The SVG does not look right

- Open the SVG in a browser first
- Try another render setting
- Make sure the scene uses the view you want
- Save the file again with a new name

### Windows blocks the file

- Right-click the file
- Open Properties
- If you see an Unblock option, turn it on
- Run the app again
